My Projects

I have worked on various real-time projects in collaboration with multiple companies.

User-Specific Safe Route Recommendation System

In this project we proposed a User-Specific Safe Route Recommendation System which provides safe route on maps to the user based on past criminal records for a geographical region. Our approach is implemented at two levels first to realize the user-specific features and the latter to actuate the safe route generation using a set of e cient algorithms. We visualize the determined routes through a colour-code based map user interface.

Facial Expression Recognition Software

This project aims to present an approach to recognize human facial expressions as part of Human Computer Interaction. It is intended on recognizing specific generic expressions and accurately extracting the useful emotional features by employing Bezier curves approximation technique. The implementation was accompanied by a well-defined software development plan and documentation.

Twitter Sentiment Analysis using Apache Spark

Social media also is an influencer in e-commerce which can provide data on people’s needs and expectations. Realizing the true feeling in a tweet is very important. Based on a series of tweets, we examine the possibility of classifying them into di erent mood classes. From each tweet, we extract di erent sets of functions, refer to a manually annotated training set, and use machine learning to perform the classification through the use of Apache Spark.

Sentiment Analysis on YouTube comments

Sentiment Analysis is the technique of using user feedback and other responses by processing it through the use of natural language processing libraries in order to identify the sentiments that the user wanted to convey and using it for feedback utilization and in creating various business propositions, schemes and improving the user experience.

E-Camera Bazaar Web Portal

The E-Camera Bazaar (ECB) web application is intended to provide complete solutions for vendors as well as customers through a single gateway using the internet as the sole medium. It will enable vendors to setup an online platform for selling their products, customers to browse through the shop and purchase them online without having to visit the shop physically
